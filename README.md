# TXAdmin-Roblox

This is a blueprint for a TXAdmin-style admin system for Roblox.

## Structure

- Server/: Server-side scripts (kick, ban, permissions)
- Client/: Client-side scripts (UI, notifications)
- Docs/: Documentation for commands and setup

## How to Use

1. Clone the repository
2. Future dev can copy files into Roblox Studio
3. Follow Docs/command_list.md for command details
